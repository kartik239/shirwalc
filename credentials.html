How to manage user accounts for Shirwal Circle.

How Accounts Work

All accounts are manually defined in js/app.js inside the USER_ACCOUNTS array. No backend. No database.

Demo Account

Email: demo@shirwalcircle.in Password: Shirwal@123 Name: Rohan Patil Phone: +91 98765 12345 Address: Datta Nagar, Shirwal

Add a New User

Find USER_ACCOUNTS in js/app.js and paste a new object:

{ username: 'newuser@example.com', password: 'User@123', name: 'Full Name', phone: '+91 90000 00000', address: 'Area, Shirwal' }

Save and re-deploy. The new user can log in immediately.

⚠️ Note

Passwords are in plain text in a static JS file. Do not use sensitive passwords. Keep this page private.
